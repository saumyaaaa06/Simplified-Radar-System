Radar Signal Processing Simulator

A simple radar signal processing simulator developed in MATLAB/GNU Octave to demonstrate the basic principles of radar target detection.

Features
- Generates a transmitted radar pulse
- Simulates multiple target echoes
- Adds Gaussian noise to the received signal
- Detects targets using peak detection
- Estimates target distances
- Visualizes transmitted and received signals

Project Structure
- Pulse Generation
- Echo Simulation
- Noise Addition
- Peak Detection
- Distance Estimation
- Signal Visualization

Output
The simulator generates:

- Transmitted Pulse
- Received Signal with Noise
- Detected Targets
